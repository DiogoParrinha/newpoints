Description

NewPoints comes with two converters, one to convert MyPlaza Turbo data to NewPoints data and one to convert MyPS data to NewPoints data.

Data converted from MyPlaza Turbo: points, forum income rate to forum rules and user group income rate to user group rules.

Data converted from MyPS: points, exempt forums to forum rules and exempt groups to user group rules.

MyPS to NewPoints

  • MAKE SURE YOU HAVE A WORKING NewPoints INSTALLATION FIRST.
  • MAKE SURE MYPS IS INSTALLED AND ACTIVATED.
  • Upload the file converters/converter2.php to the root of your MyBB installation and run it.
  • Follow the on-screen instructions.
  • Delete the file from your server once the data has been converted.
  • Uninstall MyPS.
  • Done!

MyPlaza Turbo to NewPoints

  • MAKE SURE YOU HAVE A WORKING NewPoints INSTALLATION FIRST
  • MAKE SURE MYPLAZA TURBO IS INSTALLED AND ACTIVATED
  • Upload the file converters/converter1.php to the root of your MyBB installation and run it.
  • Follow the on screen instructions.
  • Delete the file from your server once the data has been converted.
  • Uninstall MyPlaza Turbo.
  • Done!
